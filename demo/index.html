  • Hand Totals: The total value of a hand is the rightmost digit of the sum of its cards. For example, if you have a 7 and a 6, the sum is 13, so the hand value is 3. If you have a King and an 8, the sum is 8, so the hand value is 8. The goal is to get as close to 9 as possible.
